Installazione

Come installare l'APK

L'app non è ancora sul Play Store. Puoi installarla manualmente dall'APK.

  1. Vai alla release con il pulsante in alto e scarica coachpro-v*.apk dalla sezione Assets.
  2. Abilita le origini sconosciute per il browser o i file (Impostazioni → Sicurezza).
  3. Apri il file APK e conferma l'installazione.
  4. Su Xiaomi / Redmi / POCO: in Opzioni sviluppatore attiva Installa via USB (e, se serve, Installa app tramite USB).
